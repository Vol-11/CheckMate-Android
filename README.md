# CheckMate-Android

## なにこれ

- これは技育CAMPハッカソンVol.11にむけたプロダクトです。

#### 各ブランチの役割

| ブランチ名       | 用途             | 存続期間 | ベースブランチ | 命名規則           |
| ----------- | -------------- | ---- | ------- | -------------- |
| **main**    | 公開用（プロダクション相当） | 永続   | -       | `main` 固定      |
| **feature** | 機能追加・変更用       | 短命   | main    | `feature/短い説明` |
| **hotfix**  | 公開後の即時修正       | 短命   | main    | `hotfix/短い説明`  |
